# ImgurAPI-iOS
*iOS demo application to display contents using imgurAPI's data*

---
## Project Description

Basic demonstration/usage of imgur API to show the top imgur galleries from the current week. Along with each gallerie from the list, it will also contain the following data:  

1. Cover image;
2. Ups and downs balance;
3. Comment count; 
4. View count;

---

**Besides the listed basic info that the application will show, it must also fulfill the following requisites:**

1. The application must be written in Swift;
2. Correct layout defined for portrait and landscape view;
3. The app should run on all devices (iPhones and iPads) that support iOS 13.6 or higher; 
4. The user should be able to pull-to-refresh the grid;
5. An animated placeholder should be displayed on each item indicating that it is being loaded;
6. An error message should be displayed if the request to fetch the items fails;
7. A static image should be displayed on images that failed to load;
